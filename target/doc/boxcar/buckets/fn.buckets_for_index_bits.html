Function buckets_for_index_bits

Source 
pub const fn buckets_for_index_bits(bits: u32) -> usize
Expand description

Calculate the largest value of the BUCKETS generic parameter such that every element is given an index less than 2 ^ bits (though not every index less than 2 ^ bits may be given an element).

This ensures both that Index::new will reject all values greater than 2 ^ bits and that Index::into_raw will return values less than 2 ^ bits.

§Panics

bits may not exceed usize::BITS.
